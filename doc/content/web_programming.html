Веб-программирование

Фреймворк Flask (Википедия, официальный сайт) позволяет достаточно быстро создать простое web приложение.

Исходный код приложения находится в файле python\code\web\flask_app.py. Порядок запуска:

>cd python\code\web

>python flask_app.py
 * Serving Flask app 'flask_app'
 * Debug mode: off
WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on http://127.0.0.1:5000 
Press CTRL+C to quit

При запуске сервер выводит в консоль адрес по которому доступно web приложение: http://127.0.0.1:5000.

В качестве эксперимента были разработаны:

  • простая страница приветствия;
  • страница профиля пользователя;
  • страница добавления пользователя.

В качестве хранилища данных был использован словарь.
