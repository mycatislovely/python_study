# Правила оформления задачи

<!--i ["артефакты"]--> 
При решении задач курса используются следующие *артефакты* (файлы):

- файл с программой (`*.py`);
- файл с примером исходных данных (`*.txt`, суффикс `_input`);
- файл с тестом этой программы  (`*.py`, префикс `test_`);
- файл демонстрации запуска (`*.py`, суффикс `_demo`).

Некоторые файлы могут не использоваться, если это не требуется. 

Пример использования всех артифактов можно посмотреть в задаче 
[Транспонирование матрицы](<!--page 4_python_drill-->#transpond_matrix).

Сокращенный пример можно посмотреть в задаче 
[Анаграмма](<!--page 4_python_drill-->#is_anagram).
